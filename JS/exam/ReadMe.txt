Заготовка проекта!!

Перед началом требуется установить Node.js и npm!!
Все модули инициализируются с помощью файла package.json и команды "npm install".
При установке новых модулей через npm не стоит забывать прописывать "--save-dev" чтобы модули сохранялись в package.json.
Устанавливаем gulp с помощью npm глобально командой "npm i -g gulp-cli",а затем устанавливаем его в наш проект командой "npm install --save-dev gulp". 
Устанавливаем bower с помощью npm глобально командой "npm install -g bower".
Так же для работы bower требуется установить git (https://git-scm.com/downloads).

Для начала разработки проекта требуется перейти в командной строке в папку с проектом и написать gulp или gulp watch.
Для сборки готового проекта следует написать "gulp build".

При работе над проектом основные файлы для правки это index.html, main.scss, core.js.  


- Файл package.json хранит данные о проекте и все используемые модули(зависимости).
- Файл gulpfile.js хранит команды для сборщика проектов Gulp и его модулей.
- Файл .gitignore хранит данные о папках и файлах которые не должны попасть на github.
- Файл .bowerrc хранит данные о папке, в которую bower скачивает подключаемые библиотеки.

- В папке app лежат все исходники.
- В папке app/css лежат все скомпилированные и минифицированные исходники css.
- В папке app/img лежат все исходные изображения, которые при копировании в dist сжимаются.
- В папке app/js лежат все скомпилированные и минифицированные исходники js.
- В папке app/sass лежат файлы scss. В файле libs.scss расположены ссылки (import) на все css файлы подключаемых библиотек.
Данный файл объединяет все css файлы библиотек и компилится в папку css в файл libs.css, затем libs.css сжимается в libs.min.css
и подключается к проекту. В файле main.scss  пишется основной scss код.
- В папку app/libs с помощью bower устанавливаются подключаемые библиотеки. 
После установки библиотек в файле gulpfile.js требуется отредактировать "gulp.task('scripts'.." для добавления js файлов скачанных библиотек.
Так же css файлы новых бибилиотеки следует прописать в файле libs.scss("@import "../libs/sweetalert/dist/sweetalert";").    

- В папку dist копируется скомпилированный и готовый проект командой gulp build. Перед build-ом папка dist удаляется.
